 USurf Dosya Yöneticisi

USurf Dosya Yöneticisi, kullanıcıların dosyalarına göz atmalarını, indirdiklerini bulmalarını, depolama alanlarını yönetmelerini, dosyalarını taşımalarını ve çok daha fazlasını sağlayan basit bir açık kaynaklı Android dosya yöneticisi uygulamasıdır. Bu uygulamanın amacı, şaibeli arka plan işlemleri olmadan, reklamsız ve güvenilir bir kullanıcı deneyimi sunmaktır.

Özellikler

 Dosyaları kesme, kopyalama, silme gibi temel özellikler.
 FTP ile WI-FI üzerinden dosya aktarımı
 Cihaz genelinde anahtar kelime ile dosya arama
 Birden çok dosyayı birden çok yere birden çok kez kopyalayıp yapıştırma desteği
 Cihaz rootlu ise root dosyalara erişim
 Dosyaları sıkıştırma / çıkarma (Desteklenen biçimler: 7z, BZIP2, GZIP, TAR, WIM, XZ, zip)
 Ana ekrana kısayol ekleme/kaldırma desteği
 Kolay gezinme için gezinme çekmecesi
 Koyu tema desteği
 Reklam veya uygulama içi satın alma yok


Build & Çalıştır

Git deposunu klonlayın:

 git clone https://github.com/ermanergoz/USurf-file-manager.git

Projeyi Android Studio'ya aktarın:
Android Studio'yu açın. Sonra “Mevcut bir Android Studio projesini aç” seçeneğine tıklayın. Eğer Android Studio zaten açıksa, Dosya'ya ve ardından Aç'a tıklayın. Klonlanmış deponun konumunu seçin ve ardından Tamam'a tıklayın. Gradle işlemleri tamamlandıktan sonra uygulama çalışmaya hazır hale gelecektir.
