# ermanergoz.github.io

The personal portfolio of Yusuf Erman Ergoz — a software engineer with a strong focus on mobile and web development. This site showcases his work, skills, and professional background. It also hosts the privacy policies for his Android applications published on Google Play.

Visit the live site: [ermanergoz.github.io](https://ermanergoz.github.io/)

## Highlights

- Built with [React](https://react.dev/) and [TypeScript](https://www.typescriptlang.org/) for a robust, type-safe codebase.
- Powered by [Vite](https://vitejs.dev/) for blazing-fast builds and hot module replacement.
- Styled with [Tailwind CSS](https://tailwindcss.com/) and [SCSS](https://sass-lang.com/) following the BEM methodology for clean, maintainable styles.
- Fully bilingual (English / Turkish) with automatic browser language detection via [i18next](https://www.i18next.com/).
- Smooth client-side navigation with lazy-loaded routes using [React Router](https://reactrouter.com/).
- Animated code flow background driven by Web Workers for a seamless, non-blocking experience.
- WCAG 2.1 Level AA compliant — accessibility is a priority, not an afterthought.
- Fully responsive across all screen sizes with dedicated mobile and desktop layouts.

## Android Applications

Beyond web development, Erman has a proven track record in native Android development with multiple applications published on Google Play. The privacy policies for these apps are hosted directly on this site, reflecting a commitment to transparency and user trust.
